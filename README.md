# OneDrive Manager para Linux (Zorin/Ubuntu)

Instalador profissional e modular para o cliente OneDrive (abraunegg) no Linux, com suporte a sincronização automática, barra lateral do Nautilus, aliases e muito mais.

## Requisitos
- Zorin OS 17/18 ou Ubuntu 22.04/24.04 (ou derivados)
- Conexão com internet
- sudo configurado

## Instalação
```bash
git clone https://github.com/kamuizada88/OneDrive-Manager.git
cd OneDriveManager
chmod +x *.sh
./install.sh

## Autenticação
Após a instalação, execute no terminal: onedrive
